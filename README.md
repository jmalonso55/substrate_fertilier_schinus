# Sewage sludge as substrate in *Schinus terebinthifolia* Raddi seedlings commercial production

This repository contains data and partial analysis code associated with the paper entitled “Sewage Sludge as Substrate in *Schinus terebinthifolia* Raddi Seedling Commercial Production,” published in *Scientific Reports* in October 2022.

**Reference:**  
Alonso, J. M.; et al. Sewage sludge as substrate in *Schinus terebinthifolia* Raddi seedlings commercial production. *Scientific Reports*, 2022.  
https://doi.org/10.1038/s41598-022-21314-0

## Analysis reports

The rendered HTML reports include R code, outputs, tables, and figures.

- [Main analysis report](https://jmalonso55.github.io/schinus-substrate-fertilizer/)
- [Multivariate analysis report](https://jmalonso55.github.io/schinus-substrate-fertilizer/multivariate_analysis.html)

## Computational environment and reproducibility

The analyses were conducted in R using an R Markdown workflow.

The source files are available in the `analysis/` folder. These `.Rmd` files contain the narrative text, RMarkdown code, statistical analyses, tables, and figures used to generate the HTML reports.

To reproduce the reports, open the corresponding `.Rmd` file in RStudio and knit the document to HTML.

## Repository content

- `analysis/main_analysis.Rmd`: source R Markdown file for the main analysis report.
- `analysis/multivariate_analysis.Rmd`: source R Markdown file for the multivariate analysis report.
- `docs/index.html`: rendered HTML version of the main analysis report.
- `docs/multivariate_analysis.html`: rendered HTML version of the multivariate analysis report.
- `stat_aroeira.csv`: dataset used in the analyses.
- `.gitattributes`: GitHub Linguist configuration for identifying R Markdown analysis files and excluding rendered HTML files from language statistics.
- `README.md`: repository description and access information.

## Notes

This repository provides data and partial analysis code for transparency, reproducibility, and teaching purposes. The analyses presented here may not include all procedures used in the published article.
