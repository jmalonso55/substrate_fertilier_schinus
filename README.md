# Sewage sludge as substrate in *Schinus terebinthifolia* Raddi seedlings commercial production

This repository contains data and partial analysis code associated with the paper entitled “Sewage Sludge as Substrate in *Schinus terebinthifolia* Raddi Seedling Commercial Production,” published in *Scientific Reports* in October 2022.

**Reference:**  
Alonso, J. M.; et al. Sewage sludge as substrate in *Schinus terebinthifolia* Raddi seedlings commercial production. *Scientific Reports*, 2022.  
https://doi.org/10.1038/s41598-022-21314-0

## Analysis report

The rendered HTML report, including R code, outputs, tables, and figures, is available here:

[Open the analysis report](https://jmalonso55.github.io/schinus-substrate-fertilizer/)

## Repository content

- `docs/index.html`: rendered HTML report with code, outputs, and figures.
- `README.md`: repository description and access information.
- `analysis.Rmd`: source R Markdown file used to generate the HTML report.
- `stat_aroeira.csv`: dataset used in the analyses.


## Notes

This repository provides data and partial analysis code for transparency, reproducibility, and teaching purposes. The analyses presented here may not include all procedures used in the published article.
